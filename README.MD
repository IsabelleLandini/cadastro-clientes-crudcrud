# Cadastro de Clientes com CrudCrud

Aplicação web desenvolvida com HTML, CSS e JavaScript para praticar o consumo de APIs REST, organização de código e boas práticas de programação moderna.

## Funcionalidades

- Cadastro de clientes
- Listagem dinâmica de clientes
- Remoção de clientes
- Validação de formulário
- Prevenção de e-mails duplicados

## Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Programação Funcional (uso de `find` e `filter`)
- ES Modules (`import` / `export`)
- Manipulação do DOM
- Eventos com `addEventListener`
- Validação de dados

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- CrudCrud

## Métodos HTTP Utilizados

- GET → listar clientes
- POST → cadastrar cliente
- DELETE → remover cliente

## Estrutura do projeto

- `index.html` → estrutura da aplicação
- `app.js` → lógica principal
- `classes.js` → classe Cliente (POO)
- `utils.js` → funções auxiliares (validação)

## Como executar

1. Clone o repositório
2. Abra o projeto no VS Code 
3. Use o Live Server
4. Configure a URL do CrudCrud no arquivo `app.js`
