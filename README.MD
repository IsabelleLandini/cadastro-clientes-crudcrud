# Cadastro de Clientes com CrudCrud

Aplicação web desenvolvida com HTML, CSS e JavaScript para praticar o consumo de APIs utilizando Fetch API e CrudCrud.

## Funcionalidades

- Cadastrar clientes
- Listar clientes
- Excluir clientes

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Fetch API
- CrudCrud

## Métodos HTTP Utilizados

- GET
- POST
- DELETE

## Como executar

1. Clone o repositório
2. Abra o arquivo index.html ou utilize o Live Server
3. Configure uma chave válida do CrudCrud no arquivo script.js
